# 🎂 Happy Birthday Sayang

Website birthday romantis & playful untuk Sayang, dibina oleh Abi.

## ✨ Fitur

- 🧸 Tema strawberry bear (Lotso-inspired)
- 💕 Step-by-step surprise: Welcome → Game → Video → Ucapan
- 🎮 Mini game interaktif: tangkap hati & strawberry
- 🎬 Ruang video untuk video kejutan
- 💌 Ucapan dengan typewriter effect & confetti
- 🎵 Butang muzik (boleh tukar lagu)

## 🚀 Deploy di GitHub Pages (Percuma)

1. **Buat repo baru** di GitHub (contoh: `birthday-sayang`).
2. **Upload semua fail** dalam folder ini ke repo:
   - `index.html`
   - `style.css`
   - `script.js`
   - folder `assets/`
3. Pergi ke **Settings → Pages**.
4. Di bawah **Source**, pilih branch `main` dan folder `/ (root)`.
5. Klik **Save**. Dalam 1-2 minit, website akan live di:
   ```
   https://username-awak.github.io/birthday-sayang/
   ```

## 📝 Cara Tukar Kandungan

Buka `index.html` dan `script.js` untuk edit:

| Nak tukar apa | Di mana |
|---------------|---------|
| Nama "Sayang" | `index.html` — cari `Sayang` |
| Nama "Abi" | `index.html` & `script.js` — cari `Abi` |
| Ucapan panjang | `script.js` — cari `fullMessage` |
| Video | `index.html` — tukar `<source src="...">` |
| Gambar | Tukar fail `assets/placeholder-photo.svg` atau edit `index.html` |
| Lagu | `index.html` — tukar `<source src="...">` dalam `<audio>` |

## 💡 Tips

- Untuk video sendiri, upload ke YouTube sebagai **Unlisted**, kemudian guna embed code.
- Atau letak fail `video.mp4` dalam folder yang sama dan tukar `src` kepada `"video.mp4"`.
- Jangan lupa test link sebelum hari besar! 😊
